  1. Cycle 1, (external) do the contevolve! state = res_info_to_get

  1. Cycle 3, Reservoirs receive the reservoir_get(s) and respond
  2. Cycle 4, (external) reservoir_info(s) update y[], do the contevolve, state = res_info_to_send
  3. Cycle 5, (output) reservoir_info(s), (internal) state = passive
